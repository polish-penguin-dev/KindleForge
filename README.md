# KindleForge

- KindleForge Is A Simple Appstore For Jailbroken Kindles.
- It Is A Simple WAF, And Works Similarly To Mesquito.
- This Uses And Relies On *A LOT* Of Scriptlets And Icons From The Kindle Modding Community! 
- Proper Credit Is Given, But Always DM Me If You Want Something Taking Down (Or, Adding!)
